Database Overview

This database contains real healthcare invoicing data with the following structure:

Core Tables:
  • patients - Patient information
  • invoices - Invoice headers
  • invoice_details - Invoice line items
  • insurance_plans - Insurance plan data
  • service_lines - Service categories
Key Metrics:
  • Financial amounts and balances
  • Service dates and billing cycles
  • Insurance coverage details
  • Payment and adjustment tracking
Sample Queries to Get Started
Basic Queries:
  • SELECT COUNT(*) FROM patients;
  • SELECT DISTINCT ar_status FROM invoices;
  • SELECT * FROM service_lines;
Advanced Queries:
  • SELECT service_line_code, SUM(invoice_total_charges) FROM invoices GROUP BY service_line_code;
  • SELECT i.*, p.date_of_birth FROM invoices i JOIN patients p ON i.patient_id = p.patient_id;
